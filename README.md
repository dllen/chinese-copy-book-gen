# 字帖生成器（React + Bootstrap）

一个面向中小学生的字帖生成器，支持田字格/米字格/回宫格、模板字体、批量文本输入、排版参数调节、实时预览与高清 PDF 导出。内置从 `links.md` 读取的预设链接来源。

## 功能特性
- 内置诗库文库：唐诗三百首（362 首，简体）、小学语文课文（统编版 1–6 年级 479 篇）、小学英语（人教版 PEP 三年级起点 848 词 + 2574 句），按标题/作者/内容搜索，一键生成字帖
- 排版格式（考试标准）：连续排列 / 古诗格式（标题、作者居中，诗句逐句居中）/ 文章格式（标题居中、段首缩进两格、标点避头尾）/ 英文格式（按词换行不拆词，基线对齐四线三格，支持临摹空行/单词重复/中文释义）
- 文本模式：多字、多词（`|`/逗号/空格分隔）、多句（`|` 分页）、文章
- 模板字体：楷书、行书、草书、庞中华、田英章、以及自定义字体
- 网格样式：田字格、米字格、回宫格、无格（SVG 高线清晰渲染）
- 排版参数：字体大小、行距、A4/作文纸纸张格式、页边距
- 颜色/描红：文字颜色、格线颜色、描红强度、空芯效果
- 页眉与尾页填充：自定义页眉，支持尾页空格填充对齐
- 实时预览：参数变更即时刷新网格与内容
- PDF 导出：浏览器打印或高清 PDF（html2pdf）
- 本地存储：自动保存和恢复常用设置
- 无障碍与校验：中文标签与辅助说明，输入合法性校验与提示
- 单元测试：核心函数浏览器测试页

## 目录结构
- `index.html` — 页面入口，样式与外部依赖（React、Bootstrap、html2pdf）
- `app.js` — React Hooks 组件与核心逻辑
- `js/library.js` — 诗库/文库加载、搜索与选文面板
- `data/poems-tang300.json/.js` — 唐诗三百首（362 首，简体，含标题/作者/诗句/标签）
- `data/texts-xiaoxue.json/.js` — 小学语文课文（479 篇，含年级/标题/分段）
- `data/common-chars.json/.js` — 常用 1000 汉字
- `data/english-words.json/.js` — 小学英语单词（848 词，含释义/年级/单元）
- `data/english-sentences.json/.js` — 小学英语课文句子（2574 句，含年级/单元）
- `tools/build_english_data.py` — 英语数据构建脚本（单词合并 + 句子提取）
- `tools/build_data.py` — 数据构建脚本（从开源仓库拉取源数据，重新生成 data/）
- `links.md` — 预设来源链接清单（每行以 `- ` 开头）
- `tests.html` — 浏览器内置的简单单元测试页面

## 快速开始
- 方法一：双击打开 `index.html`（数据已预置为 `<script>` 加载的 JS 文件，无需任何后端/本地服务）
- 方法二：本地静态服务
  - `python3 -m http.server 8000`
  - 打开 `http://localhost:8000/`

> 数据说明：`data/*.js` 与 `data/*.json` 内容一致，页面优先读取 `<script>` 预置的 `window.__copybookData__`；重新生成数据运行 `python3 tools/build_data.py`（依赖 `opencc-python-reimplemented`）。

## 使用指南
- 诗库与文库
  - 展开「📚 唐诗三百首 / 小学语文课文」面板
  - 唐诗：按标题、作者、诗句搜索（如「李白」「明月」）；可选是否包含标题和作者
  - 课文：按标题、内容搜索，可按年级筛选；勾选要写入字帖的段落
  - 「覆盖到字帖」替换当前内容；「追加到字帖」连排追加（适合多首古诗合辑）
  - 从诗库/文库导入时自动套用对应排版格式：诗 → 古诗格式，课文 → 文章格式，英语 → 英文格式（自动切换四线三格、扩至最少 10 列）
- 小学英语
  - 展开「📚 唐诗三百首 / 小学语文课文」面板，点击「小学英语」标签
  - 支持「单词」/「句子」类型切换、按年级与单元筛选
  - 多选勾选后一键覆盖或追加到字帖；单词可勾选「附中文释义」（如 apple 苹果）
  - 英文格式自动按词换行（不拆散单词），词间一格；可选印刷体/手写体
- 排版格式
  - 连续排列：默认模式，配合「文本类型/变体」使用
  - 古诗格式：无标点的短行（标题、作者）自动居中；诗句按标点分行，每句一行居中；多首诗可连排，标题行自动识别
  - 文章格式：首行为标题（居中）；其余每行为一个自然段，段首自动缩进两格；行首/行尾标点自动避让（避头尾）
  - 英文格式：按词换行，词间一格，单词不被拆散；每个输入行另起一行，空行留空行；字母基线自动对齐四线三格第三条线（基线实线、其余虚线），大写顶到上线、g/y 等下伸到底线；可选「临摹空行」（每行后留 1~2 行空行供照着写）与「单词重复」（一词重复 1~5 次描红）；可选手写体/印刷体字体
- 文本模式与输入
  - 多字：不允许空格或标点；逐字排列
  - 多词：使用 `|`、逗号或空格分隔；词按顺序逐字铺排
  - 多句：使用 `|` 分隔分页；支持句读符号；可选“+1行/+1空行”
  - 文章：连续文本，自动按格子分页
- 模板与字体
  - 选择楷书/行书/草书/庞中华/田英章或自定义字体
  - 自定义字体需本机已安装，并使用系统字体名
- 网格与样式
  - 田字格/米字格/回宫格/无格可选；格线与文字颜色独立设置
  - 描红可选“非常深/深/较深/略浅/适中/非常浅/白色（不可见）/空芯”
- 排版参数
  - 字体大小、行距、纸张（A4 竖/横、作文纸 A4）、页边距（mm）
- 页眉与尾页设置
  - 输入页眉文字；开启“填充尾页”可补齐最后一页空格
- 预设来源（links.md）
  - 选择“从预设链接加载内容”尝试拉取文本；若受 CORS 限制不可直接加载，可点击链接在新页打开并复制
- 导出
  - “打印/另存为 PDF”使用浏览器打印对话框
  - “生成高清 PDF”使用 html2pdf 以 2x 采样导出高质量 A4 PDF

## 纸张与打印建议
- 推荐使用 Chrome 或 Edge 浏览器
- 选择“另存为 PDF”，并关闭页眉页脚（如需自定义页眉使用页面控件）

## 无障碍与校验
- 所有控制项具备中文 `label` 与必要的 `aria` 提示
- 不合法输入展示提示并禁用导出按钮

## 单元测试
- 打开 `http://localhost:8000/tests.html`
- 测试覆盖：文本解析、分页、SVG 网格生成

## 技术栈
- React 18（函数组件 + Hooks）
- Bootstrap 5 响应式布局
- html2pdf.js 高清 PDF 导出

## 预设链接规范（links.md）
- 每行一个链接，以 `- ` 开头
- 示例：
  - `- https://joyouseducation.com/word/`
  - `- https://www.an2.net/zi/`

## 备注
- 模板字体需本机已安装对应字体；未安装时将回退为系统默认字体
- 项目运行全静态，无后端依赖与敏感信息；所有数据（唐诗/课文/常用汉字）均已预置于前端，离线可用
