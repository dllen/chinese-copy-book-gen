# 快速开始指南

## 🚀 5分钟上手

### 第一步：打开应用
```bash
# 方法1：直接打开
双击 index.html

# 方法2：本地服务器（推荐）
python3 -m http.server 8000
# 访问 http://localhost:8000
```

---

## 📝 基础使用

### 1️⃣ 选择样式预设
在"打印样式"下拉框选择预设：
- **新手推荐**: 田字格标准
- **英语练习**: 四线三格标准
- **儿童练习**: 儿童卡通

### 2️⃣ 输入内容
在"输入"文本框输入汉字或词语

### 3️⃣ 生成字帖
右侧预览区域实时显示效果

### 4️⃣ 导出
- **打印**: 点击"打印/另存为PDF"
- **高清PDF**: 点击"生成高清PDF"
- **图片**: 点击"导出PNG"

---

## 🎨 快速自定义

### 修改颜色
1. 在"格子颜色"选择基础颜色
2. 或点击颜色选择器自定义

### 调整大小
- **格子尺寸**: 50-80px（建议60px）
- **字体大小**: 32-48px（建议42px）
- **行距**: 0-20px（建议8px）

### 选择字体
- 楷书（默认）
- 行书
- 庞中华
- 自定义字体

---

## 💡 常用场景

### 场景1：儿童描红
```
预设：儿童卡通
格子尺寸：70px
描红背景：适中
文字描边：细
```

### 场景2：英语练习
```
排版格式：英文格式
格子类型：四线三格
英文字体：印刷体
列数：10列以上
```

### 场景3：古诗练习
```
排版格式：古诗格式
标题/作者：自动居中
诗句：自动分行
预设：米字格标准
```

### 场景4：自定义模板
```
1. 调整所有设置
2. 点击"保存模板"
3. 输入模板名称
4. 下次直接"加载模板"
```

## 🔧 高级功能

### 配置管理
- **导出配置**: 保存当前所有设置
- **导入配置**: 从JSON恢复设置
- **重置**: 恢复默认值

### 模板管理
- **保存模板**: 保存配置+内容
- **加载模板**: 一键恢复完整设置

- **SVG格式**: 矢量输出，无限缩放
- **PDF格式**: 高清打印
- **PNG格式**: 图片分享

## ⚙️ 参数建议

### 纸张格式
- **日常练习**: A4竖版
- **横向书写**: A4横版
- **便携练习**: A5竖版
- **作文练习**: 作文纸A4

### 页边距
- **标准**: 上下16mm，左右12mm
- **自定义**: 根据需要调整

### 线条粗细
- **标准**: 1px
- **粗体**: 1.5-2px
- **细体**: 0.5px

## ❓ 常见问题

### Q1: 字体不显示？
**A**: 确保已安装对应字体，或使用楷书（系统自带）

### Q2: 打印模糊？
**A**: 使用"生成高清PDF"功能，4倍采样

### Q3: 如何批量生成？
**A**: 使用"多词"模式，用`|`分隔不同批次

### Q4: 如何保存常用设置？
**A**: 使用"导出配置"或"保存模板"功能

### Q5: 颜色不对？
**A**: 使用自定义颜色选择器输入HEX值
